# Logos

Drop one logo file per organisation here, named after its **slug**:

```
logos/<slug>.svg      (preferred)
logos/<slug>.png      (transparent or white background)
logos/<slug>.webp
```

The build matches the file to the entry automatically — no need to edit the
JSON. If no logo file exists, the dashboard shows a coloured monogram instead.

Guidelines:
- Roughly square, ~128x128px (or scalable SVG). The logo is shown in a ~30px
  rounded white tile on the cards and ~48px in the detail panel.
- Prefer logos that read on a **white** background (most brand logos do).
- Use the official brand asset. Logos are shown solely to identify each
  organisation in this ecosystem map (nominative use); trademarks remain with
  their owners. On request from an owner, a logo will be removed.
